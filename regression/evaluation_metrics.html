Introduction

# Linear Regression Evaluation Metrics :-

–In regression, our model predicts a number. So metrics tell us "how far are my predictions from actual values?"
Linear Regression Metrics Evaluation have following types:
1. Error Based MAE  ·  MSE  ·  RMSE  ·  RMSLE
2. Goodness-of-Fit R²  ·  Adjusted-R²  ·  Explained Variance
3. Percentage-Based MAPE  ·  SMAPE
4. Advanced Huber Loss  ·  Quantile Loss
A) Error Based — i

MSE (Mean Squared Error) :-

•It is average of squared absolute error.
MSE / Cost Function J(θ) J(θ) = 1/(2m) · Σi=1m (yᵢ − ŷᵢ)²
Also Written As (MAE-style note) MAE = 1/n · Σ (y − ŷ)²
•In MSE, penalizes big errors heavily.
•Units becomes squared (hard to interpret).
✓ Advantages
  • Differentiable — we can update coefficient and reach global minima.
  • Has one local & global minima.
✗ Disadvantages
  • Not Robust to outliers — Mean will try to capture outliers features.
  • Changes its units like (cm²) — hard to read.
A) Error Based — ii

MAE (Mean Absolute Error) :-

MAE Cost Function MAE = 1/n · Σi=1n | yᵢ − ŷᵢ |
✓ Advantages
  • Robust to outliers — it will not move to capture outliers.
  • Will be in same unit so easy to interpret since it is in same unit.
✗ Disadvantages
  • Convergence usually takes time.
  • Optimization is very complex.
A) Error Based — iii

RMSE (Root Mean Squared Error) :-

•Square root of MSE.
•Same units as target variable.
•Most commonly used in ML.
RMSE Formula RMSE = √MSE
MSE vs RMSE
MSE penalizes large errors more (squared).
RMSE brings units back to the original scale — easier to interpret.
When to use RMSE?
When you want interpretable error in same unit as output, and you still want to penalize large errors.
Quick Comparison:
MAE = robust, same unit, slow convergence
MSE = fast convergence, squared units, outlier-sensitive
RMSE = best of both — same unit, punishes large errors, most common
B) Goodness-of-Fit — i

R² — Coefficient of Determination :-

–R² also called the coefficient of determination.
–It measures how well your regression model explains the variability of the dependent variables.
–What R² tells you: what percentage of the variation in the output (y) is explained by all the input features (x₁, x₂, …, xₙ).
R² Formula R² = 1 − (SSres / SStot)
SSres (Residual Sum)
SSres = Σ (y − ŷ)²
Sum of squared residuals
SStot (Total Sum)
SStot = Σ (y − ȳ)²
Sum of total variation
Always SStot ≥ SSres
Interpretation:
R² = 0
Model useless
(no variance explained)
R² < 0
Worst case
(below baseline)
R² = 0.85
85% variance
explained by X ★
R² = 1
Perfect fit
(all explained)
Ex: if R² = 0.85, then 85% of the variation in y is explained by X.
R² — Limitations

Limitations of R² :-

–R² always increases when you add more predictors, even if they are irrelevant.
–This can give a false sense of improvement.
★★R² always increases when we add more feature — even if the new feature is useless.
Why this is a problem in Multiple Linear Regression:
•We may add many features → R² will keep increasing.
•But the model may become overfitting.
•Coefficients become unstable.
•Multicollinearity increases.
So R² misleads us in Multiple Linear Regression.
Solution → Use Adjusted R²
Adjusted R² corrects this by penalizing unnecessary features, so it only increases when a new predictor genuinely improves the model.
B) Goodness-of-Fit — ii

Adjusted R² :-

•It will penalize the non-correlated attributes / features.
•In R², by accounting for the no. of predictors in the model.
•It prevents overfitting of unnecessary variables.
Adjusted R² Formula Adjusted R² = 1 − ( (1 − R²)(n − 1) ) / ( n − k − 1 )

where   n = number of observations  ·  k = no. of predictors
Recall: R² = R² = 1 − SSres / SStot
✓ Why Adjusted R² is Better
  • Only increases when new predictor genuinely improves model.
  • Penalizes useless features — prevents false high R².
  • Better for Multiple Linear Regression.
✗ When NOT to use
  • For Simple Linear Regression (1 feature), R² is sufficient.
  • When comparing models with very different sample sizes.
★ Rule of Thumb:
• Use R² for simple linear regression (1 feature).
• Use Adjusted R² for multiple linear regression (2+ features).
• Adjusted R² corrects this by penalizing unnecessary features, so it only increases when a new predictor genuinely improves the model.
