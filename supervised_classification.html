Classification Algorithms

Supervised · Categorical Output
Predict a categorical label (class) from input features
Linear
Logistic Regression
Despite its name, a classification algorithm. Models the probability of class membership via the sigmoid function. Fast and interpretable on linearly separable data.
Formula
P(y=1|x) = σ(wᵀx + b) σ(z) = 1 / (1 + e⁻ᶻ) Loss = −[y·log(ŷ) + (1−y)·log(1−ŷ)]
Instance-Based
K-Nearest Neighbors
Classifies a point by majority vote among its K closest neighbors in feature space. No training phase — memorizes the dataset. Sensitive to feature scale and noise.
Formula
Class(x) = majority_vote( k nearest neighbors of x ) Euclidean: d(a,b) = √Σ(aᵢ − bᵢ)²
Tree-Based
Decision Tree Classifier
Recursively splits data on feature thresholds to maximize class purity at each node. Split criterion is Gini Impurity or Entropy (Information Gain). Prone to overfitting.
Formula
Gini: G = 1 − Σ pᵢ² Entropy: H = −Σ pᵢ · log₂(pᵢ) Info Gain = H(parent) − Σ wᵢ·H(childᵢ)
Ensemble · Bagging
Random Forest Classifier
Trains multiple trees on bootstrapped samples with random feature subsets, then aggregates by majority vote. Dramatically reduces overfitting versus a single decision tree.
Formula
ŷ = majority_vote( {T₁(x), T₂(x), ..., Tₙ(x)} ) Each tree: bootstrap sample + random feature subset
Margin-Based
Support Vector Machine
Finds the hyperplane that maximizes the margin between classes. Uses kernel functions to handle non-linearly separable data by mapping to higher-dimensional space.
Formula
Decision: f(x) = wᵀx + b Objective: min ½‖w‖² Subject to: yᵢ(wᵀxᵢ + b) ≥ 1 RBF Kernel: K(x,z) = exp(−γ‖x−z‖²)
Probabilistic
Naive Bayes
Applies Bayes' theorem assuming all features are conditionally independent given the class. Extremely fast and surprisingly effective on high-dimensional text data.
Formula
P(y|x) = P(y)·P(x|y) / P(x) Naive independence: P(x|y) = Π P(xᵢ|y) Predict: argmax_y P(y)·Π P(xᵢ|y)
Deep Learning
Neural Networks
Stacked layers of neurons learning hierarchical representations. Each neuron applies a weighted sum and activation. Trained via backpropagation and gradient descent.
Formula
Forward: aˡ = g(Wˡ·aˡ⁻¹ + bˡ) ReLU: g(z) = max(0, z) Softmax: eᶻⁱ / Σeᶻʲ Backprop: W ← W − α · ∂L/∂W
Ensemble · Boosting
Gradient Boosting Classifier
Builds trees sequentially where each tree corrects the residual errors of the prior ensemble. XGBoost, LightGBM, and CatBoost are production-grade implementations.
Formula
Fₘ(x) = Fₘ₋₁(x) + η · hₘ(x) Residuals: rᵢ = −∂L(yᵢ, Fₘ₋₁(xᵢ)) / ∂Fₘ₋₁ η = learning rate (shrinkage)

Evaluation Metrics

Categorical · Error Measurement
Measure how well a model predicts categorical class labels
Overall Correctness
Accuracy
Fraction of all predictions that are correct. Misleading on imbalanced datasets — a model predicting only the majority class can still score high.
Formula
Accuracy = (TP + TN) / (TP + TN + FP + FN)
Range: [0, 1] ↑ Higher is better
Positive Predictive Value
Precision
Of all instances predicted positive, how many actually are? High precision means fewer false alarms. Important when false positives are costly (e.g. spam filters).
Formula
Precision = TP / (TP + FP) TP = True Positives FP = False Positives
Range: [0, 1] ↑ Higher is better
Sensitivity · TPR
Recall
Of all actual positives, how many were correctly detected? Critical when false negatives are costly — e.g. cancer screening or fraud detection systems.
Formula
Recall = TP / (TP + FN) FN = False Negatives Also called: Sensitivity / TPR
Range: [0, 1] ↑ Higher is better
Harmonic Balance
F1-Score
Harmonic mean of Precision and Recall. Balanced metric for imbalanced datasets where both false positives and false negatives carry real consequences.
Formula
F1 = 2 × (Precision × Recall) / (Precision + Recall) Penalizes imbalance between P & R
Range: [0, 1] ↑ Higher is better
Error Breakdown
Confusion Matrix
Table of actual vs. predicted class counts across all categories. Reveals which classes are confused. Foundation for Precision, Recall, F1, and Accuracy.
Formula
Pred + Pred − Actual + | TP | FN | Actual − | FP | TN | Diagonal = correct predictions Off-diag = error types
N×N matrix Diagonal dominant = better
Threshold-Independent
ROC-AUC Score
Area under the ROC curve (TPR vs FPR across all thresholds). AUC=1 is perfect; AUC=0.5 is random guessing. Robust to class imbalance.
Formula
TPR = TP / (TP + FN) ← y-axis FPR = FP / (FP + TN) ← x-axis AUC = ∫₀¹ TPR d(FPR) AUC = 0.5 → random | 1.0 → perfect
Range: [0, 1] ↑ Higher is better
Probabilistic Loss
Log Loss (Cross-Entropy)
Penalizes confident wrong predictions heavily. Measures uncertainty in probabilistic outputs. Used as the training loss in logistic regression and neural networks.
Formula
Log Loss = −1/n · Σ[y·log(ŷ) + (1−y)·log(1−ŷ)] yᵢ = true label (0 or 1) ŷᵢ = predicted probability
Range: [0, ∞) ↓ Lower is better
Balanced Binary Metric
Matthews Correlation Coeff.
Single balanced metric for binary classification accounting for all four confusion matrix cells. Reliable even under severe class imbalance.
Formula
MCC = (TP·TN − FP·FN) / √((TP+FP)(TP+FN)(TN+FP)(TN+FN)) +1 = perfect | 0 = random | −1 = inverse
Range: [−1, 1] ↑ Higher is better
