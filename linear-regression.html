Example Formula

\[ y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_p x_p \]
\[ RSS = \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 \]
\[ J(m,c)=\frac{1}{2m} \sum (y - \hat{y})^2 \]
\[ m = m - \alpha \frac{\partial J}{\partial m} \] \[ c = c - \alpha \frac{\partial J}{\partial c} \]
\[ Cost = RSS + \lambda \sum_{j=1}^{p} |\beta_j| \]
