‚ùó What Is Multicollinearity?

Multicollinearity occurs when two or more independent variables are highly correlated.

‚ö†Ô∏è Why is it a problem?
  • Coefficients become unreliable
  • Model becomes unstable
  • p-values become misleading
  • Hard to interpret variable effects
üß™ How to Detect Multicollinearity

‚úîÔ∏è Variance Inflation Factor (VIF)

VIF > 5 or 10 ‚Üí high multicollinearity.

üõ†Ô∏è How to Handle Multicollinearity
  1. Remove correlated features
  2. PCA
  3. Ridge/Lasso
  4. Combine correlated features
üßë‚Äçüíª Python Coding Example (Detect & Fix Multicollinearity)
Step 1: Import Libraries
import pandas as pd
import numpy as np
from sklearn.linear_model import LinearRegression
from statsmodels.stats.outliers_influence import variance_inflation_factor
from sklearn.preprocessing import StandardScaler
üìå Example Dataset
data = {
    'X1': [10, 20, 30, 40, 50],
    'X2': [11, 21, 31, 41, 51],   # X2 highly correlated with X1
    'X3': [5, 3, 6, 2, 1],
    'Y' : [100, 200, 300, 400, 500]
}

df = pd.DataFrame(data)
‚úÖ Step 2: Calculate VIF
X = df[['X1', 'X2', 'X3']]
vif_data = pd.DataFrame()
vif_data["Feature"] = X.columns
vif_data["VIF"] = [variance_inflation_factor(X.values, i)
                   for i in range(X.shape[1])]
print(vif_data)
Feature   VIF
X1       251.2
X2       259.4
X3         1.8

üëâ X1 and X2 have very high VIF.

‚úîÔ∏è Step 3A: Remove One Feature
X_new = df[['X1', 'X3']]
model = LinearRegression()
model.fit(X_new, df['Y'])
print("Coefficients:", model.coef_)
‚úîÔ∏è Step 3B: Ridge Regression (recommended)
from sklearn.linear_model import Ridge

ridge = Ridge(alpha=1.0)
ridge.fit(X, df['Y'])

print("Ridge Coefficients:", ridge.coef_)
‚úîÔ∏è Step 3C: PCA
from sklearn.decomposition import PCA
