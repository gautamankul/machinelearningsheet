Regression Algorithms

Supervised · Continuous Output
Predict a continuous numeric value from input features
Linear
Linear Regression
Models feature–target relationships as a linear function. Minimizes the sum of squared residuals (OLS). Assumes linearity, independence, and homoscedasticity.
Formula
ŷ = w₀ + w₁x₁ + ... + wₙxₙ = wᵀx + b OLS Cost: J = Σ(yᵢ − ŷᵢ)² / 2m Closed form: w = (XᵀX)⁻¹ Xᵀy
Linear (Extended)
Polynomial Regression
Extends linear regression with polynomial feature terms, enabling non-linear relationships while keeping a linear model under the hood.
Formula
ŷ = w₀ + w₁x + w₂x² + ... + wₙxⁿ φ(x) = [1, x, x², ..., xⁿ] Fit: ŷ = wᵀφ(x)
Regularized · L2
Ridge Regression
Adds an L2 penalty on weights to prevent overfitting. Shrinks coefficients toward zero but rarely eliminates them entirely. Ideal for multicollinear data.
Formula
J = Σ(yᵢ − ŷᵢ)² + λΣwⱼ² Closed form: w = (XᵀX + λI)⁻¹ Xᵀy
Regularized · L1
Lasso Regression
L1 regularization that can shrink coefficients to exactly zero, performing automatic feature selection. Solved via coordinate descent — no closed form.
Formula
J = Σ(yᵢ − ŷᵢ)² + λΣ|wⱼ| Coordinate descent (no closed form) Sparse w → automatic feature selection
Margin-Based
Support Vector Regression
Fits a function within an ε-tube around the data — points inside incur no loss. Uses the kernel trick for non-linear regression on complex distributions.
Formula
Minimize: ½‖w‖² + C·Σ(ξᵢ + ξᵢ*) Subject to: yᵢ − wᵀxᵢ − b ≤ ε + ξᵢ wᵀxᵢ + b − yᵢ ≤ ε + ξᵢ*
Tree-Based
Decision Tree Regression
Partitions feature space into rectangular regions and predicts the mean target value in each leaf. Splits minimize MSE at each node. Prone to overfitting.
Formula
Split: argmin_t [MSE(left) + MSE(right)] MSE = Σ(yᵢ − ȳ)² / n Leaf prediction: ŷ = mean(y in leaf)
Ensemble · Bagging
Random Forest Regression
Ensemble of decision trees trained on bootstrapped samples with random feature subsets. Final prediction is the average of all trees — reduces variance greatly.
Formula
ŷ = mean( {T₁(x), T₂(x), ..., Tₙ(x)} ) Variance: Var(avg) ≈ σ²/n (assuming uncorrelated trees)
Ensemble · Boosting
Gradient Boosting
Iteratively adds weak trees to correct residuals. XGBoost extends this with second-order Taylor expansion of the loss and L2 tree regularization.
Formula
F₀(x) = mean(y) Fₘ(x) = Fₘ₋₁(x) + η · hₘ(x) Residual: rᵢ = yᵢ − Fₘ₋₁(xᵢ) hₘ fits residuals at each step

Evaluation Metrics

Continuous · Error Measurement
Measure how close predicted values are to actual numeric values
Absolute Error
MAE — Mean Absolute Error
Average of absolute differences between predictions and actuals. Robust to outliers. Interpretable — error is in the same unit as the target.
Formula
MAE = (1/n) · Σ |yᵢ − ŷᵢ| Linear penalty per error
Range: [0, ∞) ↓ Lower is better
Squared Error
MSE — Mean Squared Error
Average of squared differences. Amplifies large errors — useful when big mistakes must be penalized heavily. Default loss in linear regression.
Formula
MSE = (1/n) · Σ (yᵢ − ŷᵢ)² Quadratic penalty → outlier-sensitive
Range: [0, ∞) · Unit: target² ↓ Lower is better
Scaled Squared Error
RMSE — Root Mean Squared Error
Square root of MSE — restores error to the original unit scale. More interpretable than MSE while still penalizing large errors more than MAE.
Formula
RMSE = √[ (1/n) · Σ (yᵢ − ŷᵢ)² ] = √MSE
Range: [0, ∞) ↓ Lower is better
Explained Variance
R² — Coefficient of Determination
Proportion of variance in the target explained by the model. R²=1 is a perfect fit; R²=0 means no better than predicting the mean.
Formula
R² = 1 − SS_res / SS_tot SS_res = Σ(yᵢ − ŷᵢ)² ← residuals SS_tot = Σ(yᵢ − ȳ)² ← total variance
Range: (−∞, 1] ↑ Higher is better
Penalized Variance
Adjusted R²
Penalizes R² for adding features that don't genuinely improve the model. Prevents inflation from irrelevant predictors. Preferred for multi-feature models.
Formula
R²_adj = 1 − (1−R²)·(n−1)/(n−p−1) n = samples · p = predictors Increases only if new features help
Range: (−∞, 1] ↑ Higher is better
Relative Error
MAPE — Mean Absolute % Error
Expresses error as a percentage of actual values — useful when comparing across datasets of different scales. Undefined when actuals are zero.
Formula
MAPE = (100/n) · Σ |yᵢ − ŷᵢ| / |yᵢ| Unit: percentage (%)
Range: [0, ∞) ⚠ Undefined if yᵢ = 0
Hybrid Loss
Huber Loss
Combines MSE and MAE. Acts like MSE for small errors (sensitive, differentiable) and like MAE for large errors (outlier-robust). Threshold δ is a hyperparameter.
Formula
Lδ(r) = { ½r² if |r| ≤ δ ← MSE zone δ·|r| − ½δ² if |r| > δ ← MAE zone } r = yᵢ − ŷᵢ
Hyperparameter: δ Robust + smooth
Worst-Case Error
Max Error
The single worst prediction error in the dataset. Critical in safety-critical systems where even one large deviation can have serious consequences.
Formula
Max Error = max( |yᵢ − ŷᵢ| ) Captures the single worst prediction
Range: [0, ∞) Safety-critical use
