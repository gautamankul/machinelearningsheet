Regression Algorithms

8 algorithms
🎯 Predict a continuous numeric value from input features
Linear Regression
Linear
Models the relationship between features and a continuous target as a linear function. Minimizes the sum of squared residuals (OLS). Assumes linearity and independence.
📐 Formula / Equation
ŷ = w₀ + w₁x₁ + ... + wₙxₙ = wᵀx + b OLS Cost: J = Σ(yᵢ - ŷᵢ)² / 2m Closed form: w = (XᵀX)⁻¹ Xᵀy
Polynomial Regression
Linear (extended)
Extends linear regression by adding polynomial terms of features, allowing it to capture non-linear relationships while still using a linear model.
📐 Formula / Equation
ŷ = w₀ + w₁x + w₂x² + ... + wₙxⁿ Feature transform: φ(x) = [1, x, x², ..., xⁿ] Fit: ŷ = wᵀφ(x)
Ridge Regression (L2)
Regularized
Linear regression with L2 regularization. Penalizes large weights to prevent overfitting, shrinking coefficients toward zero but rarely to exactly zero.
📐 Formula / Equation
J = Σ(yᵢ - ŷᵢ)² + λΣwⱼ² Closed form: w = (XᵀX + λI)⁻¹ Xᵀy λ = regularization strength
Lasso Regression (L1)
Regularized
Linear regression with L1 regularization. Unlike Ridge, Lasso can shrink coefficients exactly to zero, performing automatic feature selection.
📐 Formula / Equation
J = Σ(yᵢ - ŷᵢ)² + λΣ|wⱼ| Solved via coordinate descent (no closed-form solution) Sparse w → automatic feature selection
Support Vector Regression (SVR)
Margin-based
Fits a function within an ε-tube around the data. Points inside the tube incur no loss. Uses kernel trick for non-linear regression.
📐 Formula / Equation
Minimize: ½‖w‖² + C·Σ(ξᵢ + ξᵢ*) Subject to: yᵢ - wᵀxᵢ - b ≤ ε + ξᵢ wᵀxᵢ + b - yᵢ ≤ ε + ξᵢ*
Decision Tree Regression
Tree-based
Partitions feature space into regions and predicts the mean of training samples in each region. Splits are chosen to minimize variance (MSE) in child nodes.
📐 Formula / Equation
Split criterion: MSE = Σ(yᵢ - ȳ)² / n Leaf prediction: ŷ = mean(y in leaf) Split: argmin_t [MSE(left) + MSE(right)]
Random Forest Regression
Ensemble
Ensemble of decision tree regressors trained on bootstrapped samples with random feature subsets. Final prediction is the average of all tree outputs.
📐 Formula / Equation
Prediction = mean( {Tree₁(x), Tree₂(x), ..., Treeₙ(x)} ) Variance reduction: Var(avg) ≈ σ²/n (uncorrelated trees)
Gradient Boosting Regression
Ensemble
Iteratively adds trees to correct residuals of the current model. XGBoost adds second-order Taylor expansion of the loss for faster, more accurate convergence.
📐 Formula / Equation
F₀(x) = mean(y) Fₘ(x) = Fₘ₋₁(x) + η·hₘ(x) Residual: rᵢ = yᵢ - Fₘ₋₁(xᵢ) hₘ fits residuals XGBoost adds L2 tree regularization

Evaluation Metrics

8 metrics
🎯 Measure how close predicted values are to actual numeric values
MAE — Mean Absolute Error
Average of absolute differences between predicted and actual values. Robust to outliers. Error is in the same unit as the target variable.
📐 Formula / Equation
MAE = (1/n) · Σ |yᵢ - ŷᵢ| Linear penalty for each error Range: [0, ∞) Lower = better Unit: same as target
MSE — Mean Squared Error
Average of squared differences. Amplifies large errors due to squaring — useful when big mistakes must be penalized more. Default loss in Linear Regression.
📐 Formula / Equation
MSE = (1/n) · Σ (yᵢ - ŷᵢ)² Quadratic penalty → sensitive to outliers Range: [0, ∞) Lower = better Unit: target²
RMSE — Root Mean Squared Error
Square root of MSE. Returns error to the original unit scale. More interpretable than MSE while still penalizing large errors more than MAE.
📐 Formula / Equation
RMSE = √[ (1/n) · Σ (yᵢ - ŷᵢ)² ] RMSE = √MSE Range: [0, ∞) Lower = better Unit: same as target
R² — Coefficient of Determination
Measures the proportion of variance in the target explained by the model. R²=1 is a perfect fit; R²=0 means no better than predicting the mean.
📐 Formula / Equation
R² = 1 - SS_res / SS_tot SS_res = Σ(yᵢ - ŷᵢ)² ← residuals SS_tot = Σ(yᵢ - ȳ)² ← total variance Range: (-∞, 1] Higher = better
Adjusted R²
Penalizes R² for adding features that don't improve the model. Prevents artificially inflating R² by adding irrelevant predictors. Preferred for multi-feature models.
📐 Formula / Equation
R²_adj = 1 - (1-R²)·(n-1)/(n-p-1) n = number of samples p = number of predictors Increases only if new features genuinely help
MAPE — Mean Absolute Percentage Error
Expresses error as a percentage of actual values. Useful for comparing model performance across datasets with different scales. Undefined when actual values are zero.
📐 Formula / Equation
MAPE = (100/n) · Σ |yᵢ - ŷᵢ| / |yᵢ| Unit: percentage (%) Range: [0, ∞) Lower = better Caution: undefined if yᵢ = 0
Huber Loss
Hybrid of MSE and MAE. Behaves like MSE for small errors (sensitive, differentiable) and like MAE for large errors (robust to outliers). Controlled by threshold δ.
📐 Formula / Equation
Lδ(r) = { ½r² if |r| ≤ δ ← MSE zone δ·|r| - ½δ² if |r| > δ ← MAE zone } r = yᵢ - ŷᵢ δ = threshold hyperparameter
Max Error
The worst-case prediction error — largest absolute deviation between any predicted and actual value. Critical in safety systems where no single prediction can be far off.
📐 Formula / Equation
Max Error = max(|yᵢ - ŷᵢ|) Captures the single worst prediction Range: [0, ∞) Lower = better Use: safety-critical applications
